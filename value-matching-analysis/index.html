This interaction was an introduction to the differences between jumps in value. The goaL was the make the outer colors have equal jumps in value from one to the next. The inner colors were to follow the same pattern but also keep the value boundary between the outer and inner colors appearing the same hardness in each quadrant. This was an optical exercise working within the same hue and different values.
