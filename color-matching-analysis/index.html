This project acted as an introduction to color sensitivity especially when translating between different printing processes. In the first step, it was important to be sensitive to hue, value and saturation between a specific spot of the base image and the Color-Aid swatch. Many swatches were close but it was important to find an exact match to ensure print quality. From there subsequent print tests on the laser printer encouraged the careful matching of these laser prints to the single color swatches. Because laser printing is composed of four colors and the swatches were made with one solid color, simply eyedropping the color from one to the other did not provide an exact match. Grids of slightly varying hues and values were important in determing the closest print match.
